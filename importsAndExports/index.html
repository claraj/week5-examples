Modules in client-side JavaScript

This page has two JavaScript files, script.js and data.js. Script.js is the main code file, and it would like to import data.js.

The following data was read from a module by the script file,

  • Include only script.js and include type="module attribute in the script tag.
  • data.js needs to export whatever other script files will need, examples https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Statements/export
  • script.js will import whatever it needs from data.js. Examples https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Statements/import
  • To view this page, you'll need to serve it from a web server. WebStorm will do this for you, when you open a page from the WebStorm IDE using the browser icons in the top left.
  • Not using WebStorm? Install Node.js and then run npx http-server from the directory your HTML file is in, to start a basic HTTP server. You'll see output from the command with a URL you can use to view your page. It will be easier if your file is already called index.html otherwise add the html file name to the end of the URL.
